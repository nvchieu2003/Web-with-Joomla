Configuration

Onglet "Params"

Dates display : Contient le formatage des dates en PHP (voir ici)
Invoice number : Prochain numéro de facture qui sera appliqué.
Invoice format : Nommage des factures, affichage du nom ainsi formaté dans la vue "Invoices"
Order number : Prochain numéro de commande qui sera appliqué.
Order format : Nommage des commandes, affichage du nom ainsi formaté dans la vue "Invoices".
Quote number : Prochain numéro de devis qui sera appliqué.
Quote format : Nommage des devis, affichage du nom ainsi formaté dans la vue "Quotes".
Admin Email/Admin Name : Les mails à l'administrateur caractérisés par la présence du mot clé [Admin] dans la liste des Emails sont envoyés à l'adresse écrite dans le champ "Admin Email" avec le nom "Admin Name". Il est conseillé de les remplir même s'ils sont présents dans la configuration de votre site Joomla!.

Pour les utilisations avancées, nous permettons la personnalisation des données générées dans l'export des factures au format CSV, ainsi que le templating des PDF et du HTML pour les factures différemment des commandes. Les champs concernés sont montrés ci-après.

Les tags sont traduisibles dans les fichiers de langue, pour les rendre traduisibles il faut que vous les placiez entre crochets comme ceci : [invoice_number] sera traduit par "Numéro de facture" si vous êtes en français.

Onglet "Community Builder"

Contient les infos nécessaire à l'intégration avec Community Builder ainsi que la possibilité de le désactiver dans Invoicing.

If you enable Community Builder, the fields "Adress field", "City Field" and "Country field" will be filled automatically.

Onglet "Javascript"

The default setting for jquery is "YES", if you choose "NO", you have to be sure that another component, plugin or template that loads Jquery to be able to use AdsManager in order to not loads more than once Jquery and keep the best performance and celerity for your website.
